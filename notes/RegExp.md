# RegExp

# 기본 문법

## 1️⃣ 그룹 및 문자 집합

| **Chracter** | **뜻** | Ex |
| --- | --- | --- |
| `|` | 또는 |  |
| `()` | 그룹 |  |
| `[]` | 문자셋, 괄호안의 어떤 문자든 | - `/gr[aed]y/` :a,e,d 
- `/gr[a-f]y/` : a~f 
- `/[A-Za-z0-9]/`: 알파벳 또는 숫자  |
| `[^]` | 부정 문자셋, 괄호안의 어떤 문가 아닐때 | - `/[^A-Za-z0-9]`: 알파벳 또는 숫자 제외  |
| `(?:)` | 찾지만 기억하지는 않음 | - `/gr(e|a)y/` : e와 a가 같은 그룹으로 지정
- `/gr(?:e|a)y/` : 따로 그룹이 만들어지지 않음
-두 예시 모두 gr로 시작, e 또는 a 포함, y로 끝나는 패턴  |

## 2️⃣ 수량

| **Chracter** | **뜻** | Ex |
| --- | --- | --- |
| `?` | 1개 이하 (0개 또는 1개) | - `/gra?y/` : a가 없거나 1개 있거나
⇒ gry, gray |
| `*` | 0개 이상 | - `/gra*y/` : a가 없거나 1개 이상
⇒ gry, gray, graay, graaay… |
| `+` | 1개 이상 | `/gra+y/` : a가 1개 이상
⇒ gray, graay, graaay… |
| `{n}`
 | n번 반복 | -`/gra{3}y/`: a가 3개 
⇒ graaay |
| `{min,max}` | 최소, 그리고 최대 | -`/gra{2,}y/`:a가 2개이상 
⇒ graay, graaay, graaaay…

-`/gra{2,3}y/`:a가 2개이상 3개 이하 
⇒ graay, graaay |

## 3️⃣ 경계 유형

| **Chracter** | **뜻** | Ex |
| --- | --- | --- |
| `\b` | 단어 경계 | - `/\bYa/` : 단어의 시작에 있는 Ya
- `/Ya\b/` : 단어의 끝에 있는 Ya |
| `\B` | 단어 경계가 아님 | -`/\BYa/` : 단어의 시작에 있지 않은 Ya 
⇒ 단어의 중간 또는 끝에 있는 Ya

- `/Ya\B/` : 단어의 끝에 있지 않은 Ya
⇒ 단어의 시작 또는 중간에  있는 Ya |
| `^` | 문장의 시작 | - `/^Ya/` : 문장의 시작에 있는 Ya |
| `$` | 문장의 끝 | - `/Ya$/` : 문장의 끝에 있는 Ya |

## 4️⃣ 문자 클래스

| **Chracter** | **뜻** | Ex |
| --- | --- | --- |
| `\` | 특수 문자 그대로 | - `/[\.\?]/`:  .또는 ?  |
| `.` | 어떤 글자 (줄바꿈 문자 제외) |  |
| `\d` (digit ) | 숫자 |  |
| `\D` | 숫자 아님 |  |
| `\s` (space) | 공백 |  |
| `\S` | 공백 아님 |  |

# `문자열.match(RegExp)`

> `[ 전체매치, 그룹1, 그룹2, ... ]` 순서의 배열을 return한다.
> 

### Ex)

```jsx
"Hello World".match(/(Hi|Hello) (World|Earth)/)

//결과
[
  "Hello World", // ← 배열[0]: 전체 매치
  "Hello",       // ← 그룹1
  "World"        // ← 그룹2
]
```
